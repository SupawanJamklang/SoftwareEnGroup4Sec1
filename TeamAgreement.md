Teamagreement

การสื่อสาร
1.	ทุกคนในทีมสื่อสารพูดคุยกันผ่าน Discord และ Facebook Messenger
2.	ถ้าหากมีการนัดหมายกันถ้าไม่สามารถร่วมประชุมหรือทำงานได้ต้องทำการแจ้งล่วงหน้าก่อนถึงเวลานัดหมาย
การทำงานร่วมกัน
1. ทุกคนต้องมีส่วนร่วมในการตัดสินใจและวางแผนการทำงานในงานกลุ่มแต่ละงานที่ได้รับมอบหมาย
2. ทุกคนในทีมจะต้องรับผิดชอบต่องานที่ได้รับมอบหมายและส่งมอบงานตามที่ตกลงกันไว้ทุกครั้ง

ความซื่อสัตย์และความเคารพ
1.	ทุกคนในทีมจะต้องเคารพการตัดสินใจซึ่งกันและกัน 
2.	มีการเคารพความคิดเห็นและข้อเสนอแนะของสมาชิกภายในทีม
3.	ทุกคนในทีมจะต้องมีความซื่อสัตย์ในการทำงานและการสื่อสารงาน

การจัดการไฟล์
1. ทุกคนในทีมจะต้องเขียนความคิดเห็นหรือคำอธิบายทุกครั้งที่มีการ commit
2. ทุกคนในทีมจะต้องจัดการไฟล์และโฟลเดอร์ให้เป็นระเบียบเรียบร้อยทุกครั้ง
3. ทุกคนในทีมจะต้องตรวจสอบความถูกต้องของไฟล์ทุกครั้งก่อนที่จะทำการ merge หรือ push ขึ้น Repository
4. ทุกคนในทีมจะต้องช่วยกันให้ถึงที่สุด

การเขียนโค้ด
1. ทุกคนใช้ชื่อตัวแปรหรือฟังก์ชันที่สื่อความหมายและสอดคล้องกับงานที่ทำ
2. ชื่อ Class ขึ้นต้นด้วยตัวพิมพ์ใหญ่
3. ชื่อ Method , ตัวแปร repository , service ขึ้นต้นด้วยพิมพ์เล็ก คั่นระหว่างคําด้วย พิมพ์ใหญ่ 
ex addAddress, addressRepository
